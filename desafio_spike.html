    • desviacion_cat: una variable categrica que vale 1si la desviacin_pct absoluta es mayor a 15% (esto es, si es menor a -15% o mayor a 15%)

Al crear la nueva variable 'desviacion_pct' aparecen valores infinitos, que provienen de cuando el costo en dolares de una barra en cierto momento del tiempo es igual a 0, habiendo un costo programado para dicha barra en ese momento. from matplotlib import dates
def time_plot_costo_barra(codigo_barra, fecha_inicial, fecha_final):
    df = costo_marginal[(costo_marginal.barra_mnemotecnico == codigo_barra) & (costo_marginal.fecha_hora >= fecha_inicial) & (costo_marginal.fecha_hora <= fecha_final)]
    df_melt = pd.melt(df, id_vars = ['fecha_hora', 'barra_mnemotecnico'], value_vars = ['costo_en_dolares', 'costo_prog'])
    fig, ax = plt.subplots()
    p = sns.lineplot(x="fecha_hora", y="value",
             hue="variable",
             data=df_melt)
    ax.xaxis.set_major_formatter(dates.DateFormatter("%d-%m"))
    plt.xticks(rotation=45)
    return p

# Esta funcin fue extrada de https://www.kaggle.com/lokeshkumarn/timeseries-multivariate
def series_to_supervised(data, n_in=1, n_out=1, dropnan=True):
    n_vars = 1 if type(data) is list else data.shape[1]
    df = pd.DataFrame(data)
    cols, names = list(), list()
    # secuencia input (t-n, ... t-1)
    for i in range(n_in, 0, -1):
        cols.append(df.shift(i))
        names += [('var%d(t-%d)' % (j+1, i)) for j in range(n_vars)]
    # secuencia para el forecast (t, t+1, ... t+n)
    for i in range(0, n_out):
        cols.append(df.shift(-i))
        if i == 0:
            names += [('var%d(t)' % (j+1)) for j in range(n_vars)]
        else:
            names += [('var%d(t+%d)' % (j+1, i)) for j in range(n_vars)]
    # unir todo
    agg = pd.concat(cols, axis=1)
    agg.columns = names
    # Botar NA's, aunque ya las rellenamos con 0's asi es que este paso se debera poder omitir
    if dropnan:
        agg.dropna(inplace=True)
    return agg

8. Reflexión¶

  • Por qu sera bueno utilizar un modelo como este para anticiparse a desvos de precios de la energa?

El Sistema Elctrico Nacional al parecer es el encargado de regular la generacin, produccin y transferencia de energa elctrica en el pas. Dentro de sus objetivos est programar la generacin de electricidad para minimizar costos al mismo tiempo que se vela por proveer el acceso a sistemas de transmisin. De esta forma, cuando las empresas privadas no se ajustan a la generacin y costos programados, deben pagar multas. As, este sistema podra ayudar al Sistema Elctrico Nacional para minimizar costos.

  • Qu casos de uso te imaginas podran beneficiarse teniendo acceso a un modelo como este?

Adems de ayudar al Sistema Elctrico Nacional a reducir costos, tambin un modelo como este podra servir a las empresas privadas para detectar qu les est generando el desvo en sus propios compromisos programticos. Otra posible aplicacin se relaciona con que es posible que el desvo tenga una repercusin en cmo se distribuye la energa por parte del Coordinador Elctrico Nacional, si es el caso, un modelo de este estilo podra servir para mantener una distribucin ptima y eficiente al adelantarse al comportamiento de las empresas. Al igual que los bancos cuando detectan morosos, el Coordinador Elctrico Nacional podra detectar aquellas subestaciones que faltan a su compromiso programtico.
